### PointR2 / Rectangle
- `distance()` : √((x1-x2)² + (y1-y2)²)
- `equals()` : vrai si mêmes coordonnées.
- `surface()` : largeur × hauteur.
- `perimetre()` : 2 × (largeur + hauteur).

### Date
- Année bissextile : divisible par 4, pas par 100, sauf si divisible par 400.
- `compareTo()` : compare les dates.
- `distance()` : différence en jours.

### Jeu de dominos
Classes : Domino, EnsembleDom, ChaineDom, Joueur, Partie.

### Fractales
- Julia : z_{n+1} = z_n² + c
- Mandelbrot : z_0 = 0, même formule.
